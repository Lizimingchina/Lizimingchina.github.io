WINDOWS10的Linux子系统

摘要:在Windows10系统中，增加了一项黑科技：Linux子系统，即Windows Subsystem for Linux，以下简称”WSL“。本文主要介绍了WSL的安装使用方法，以及一些比较基础的操作。

关键词 :WSL、新手入门。

1.前言

作为刚接触Linux的新手，我们首要体验Linux的方式是通过VMVare等软件安装虚拟机，再使用图形界面操作、练习命令；或者有些爱折腾的同学会选择双系统。前者通常比较占用内存资源，后者两个系统来回切换，也比较麻烦。而Windows Subsystem for Linux (以下简称WSL)为我们提供了一种新的选择。它能够在比较小的资源占用下，让我们体验到原生的Linux功能。本文将从安装入手，用图文一步步说明，起到一定的参考作用。
